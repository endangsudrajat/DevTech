## 🚀 Overview

**DevTech Portfolio** is a meticulously crafted personal website for developers who want to showcase their work with a premium aesthetic. Featuring a **Bento Grid** layout, this template prioritizes visual hierarchy and interactive user experience.

Developed by **Endang**, a Lead Web Developer, this project demonstrates advanced front-end techniques, including glassmorphism, parallax effects, and smooth state transitions.

## ✨ Key Features

- 🏗️ **Modern Bento Layout** – A clean, organized grid system that scales beautifully across all devices.
- 🌓 **Dynamic Theme Switching** – Seamless Dark/Light mode toggle with persistent state.
- 🎨 **Glassmorphism UI** – Modern design system with subtle transparencies and interactive "shine" effects.
- ⌨️ **Interactive Typing Effect** – Engaging "About Me" section with a dynamic typewriter animation.
- 🖱️ **Advanced Tilt Effects** – 3D parallax hover interactions on profile cards using Vanilla JavaScript.
- 📱 **Fully Responsive** – Optimized for Desktop, Tablet, and Mobile with fluid typography.
- ⚡ **Performance Optimized** – Built with Vanilla JS and CSS for lightning-fast load times.
- 📋 **Copy-to-Clipboard** – One-click email copying for better UX.

## 🛠️ Tech Stack

- **Core:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons:** [Font Awesome 6](https://fontawesome.com/)
- **Typography:** [Google Fonts](https://fonts.google.com/) (Inter & Outfit)
- **Animations:** CSS Keyframes & JavaScript Intersection Observer patterns
